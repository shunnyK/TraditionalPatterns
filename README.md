# HeritageAI

## AI 기반 개인 맞춤형 전통문양 생성 서비스

---

## 1. 프로젝트 소개

HeritageAI는 사용자의 감정, 목적, 취향을 분석하여 한국 전통문양 데이터를 기반으로 개인 맞춤형 전통문양을 생성하는 AI 서비스입니다.

기존의 전통문양 데이터는 대부분 단순 열람 형태로 제공되고 있어 일반 사용자가 문양의 의미와 상징성을 이해하고 활용하기 어려웠습니다.

HeritageAI는 전통문양 데이터와 생성형 AI를 결합하여 누구나 자신의 감정과 이야기를 한국 전통문화의 상징으로 표현할 수 있도록 지원합니다.

---

## 2. 문제 정의

- 전통문양 데이터는 개방되어 있으나 활용성이 낮음
- 전통문양의 의미와 상징성이 대중에게 잘 전달되지 않음
- 생성형 AI 서비스는 서양 스타일 중심 결과물이 대부분임
- 한국 전통문화 데이터를 활용한 개인 맞춤형 콘텐츠 생성 서비스가 부족함

---

## 3. 활용 데이터

### 한국문화정보원 전통문양 데이터

총 22,000건

활용 정보

- meaning (상징 의미)
- emotion (감성 표현)
- motif (모티프)
- form (형태 특징)
- patern_type (문양 분류)
- patern_usage (활용 분야)

예시

- 장수
- 행복
- 번영
- 고결함
- 사랑

등의 전통적 상징 의미 포함

- 우아한

## 5. 핵심 기능

### ① 자연어 기반 감정 분석

사용자가 자유롭게 자신의 감정과 목적을 입력

예시

"부모님 환갑 선물로 드릴 건데 건강하고 행복하게 오래 사셨으면 좋겠어요."

---

### ② 전통문양 추천

입력된 감정과 목적을 기반으로 가장 적합한 전통문양 TOP3 추천

추천 기준

- 의미
- 감성
- 모티프
- 설명 텍스트

---

### ③ 의미-상징 관계 분석

22,000건 전통문양 데이터를 분석하여

의미(Meaning)와 상징(Motif)의 관계를 자동 구축

예시

- 장수 → 학, 국화, 덩굴
- 행복 → 꽃, 박쥐, 복자
- 풍요 → 모란

---

### ④ AI 전통문양 생성

추천된 문양과 의미 정보를 기반으로 생성형 AI가 새로운 전통문양 생성

반영 요소

- 의미
- 감성
- 모티프
- 형태 특징
- 문화재 활용 정보

---

### ⑤ 생성 이유 설명

AI는 생성 결과와 함께 생성 이유를 설명

예시

"장수와 행복의 의미를 표현하기 위해 학과 국화를 중심으로 문양을 구성하였습니다."

## 7. AI 활용 방식

### 추천 AI

사용자 입력과 전통문양 데이터 간 의미 기반 매칭

### 상징 분석 AI

22,000건 전통문양 데이터에서 의미-상징 관계 자동 구축

### 생성 AI

추천 결과와 상징 정보를 활용하여 새로운 전통문양 생성
